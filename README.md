# 🥭 OASIS – Exotic Mango

A **luxury minimalist marketing website** for **OASIS – Exotic Mango**, designed with a premium aesthetic, smooth animations, and high-performance architecture using modern web technologies.

---

## 🌐 Live Demo

🔗 https://shahriarxd.github.io/OASIS-EXOTIC-MANGO/

---

## ✨ Features

* 🎞️ Scroll-driven **frame-by-frame hero animation** (192 frames)
* 📱 Fully **responsive design** (mobile, tablet, desktop)
* 🌊 Smooth **parallax scrolling** and floating effects
* 🧊 Premium **glassmorphism UI components**
* ⚡ High-performance **Canvas rendering**
* 🖼️ Optimized image handling with Next.js Image
* ♿ Accessibility-focused (semantic HTML + ARIA labels)
* 🚀 Static export for ultra-fast load times
* 🎯 Clean, modern, and luxury-focused UI/UX

---

## 🛠️ Tech Stack

### Frontend

* **Next.js 14** – React framework with static export support
* **React 18** – Component-based UI architecture
* **Tailwind CSS** – Utility-first modern styling

### Animation & Interactivity

* **Framer Motion 11** – Smooth animations and transitions
* **Canvas API** – Scroll-based frame animation engine

### UI & Icons

* **Lucide React** – Lightweight and modern icon system

### Styling & Build Tools

* **PostCSS** with Autoprefixer
* **JavaScript (ES6+)**

### Deployment & Hosting

* **GitHub Pages** – Static website hosting
* **GitHub Actions** – Automated CI/CD pipeline

## 📦 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ShahriarXD/OASIS-EXOTIC-MANGO.git
cd OASIS-EXOTIC-MANGO
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Run Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

---

## 🏗️ Build for Production

```bash
npm run build
```

---

## 🚀 Deployment

This project uses **GitHub Actions** for automated deployment.

Every push to the repository:

* Triggers a build process
* Generates a static export
* Deploys automatically to GitHub Pages
